$YELLOW Keystrokes $NORM

Termmod (named TM below)  = Ctrl+Shift

Scrolling:
 Shift + Up/Down/PageUp/Pagedown: Scroll up/down
 Shift + Home/End: Scroll to top/bottom

Clipboard:
 Shift + Insert / Ctrl+Shift+y: Paste
 Ctrl+Shift+c: Copy 

TM + Up/PageUp/l: Enter lessmode. Scroll around with cursor keys.
	exit with q/ESC

Set font width/size:
  Alt+Shift +
    Insert/Delete:   Enlarge/Shrink width
    PageUp/PageDown: Zoom in / out

Scrollmarks:
  Ctrl+Alt + [0..9]: Set Scrollmark 0 - 9
  Ctrl + [0..9]:     Scroll to mark 0 - 9
